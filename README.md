## 📌 Project Overview

This repository contains my **Digital Literacy Portfolio**, created as part of the CSE0001 course.  
As a **Student Digital Ambassador**, this project aims to promote responsible, safe, and effective use of digital tools among students.

The project covers five major areas:
- Digital Awareness  
- Professional Online Presence  
- Coding & Collaboration Platforms  
- Email Etiquette & Communication  
- Cybercrime Awareness & Prevention  

Each task is designed to build essential digital skills required in academic and professional life.
